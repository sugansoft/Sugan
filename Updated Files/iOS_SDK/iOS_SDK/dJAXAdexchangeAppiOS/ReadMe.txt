 INTRODUCTION:

 	- iOS SDK version 3 package contains the set of SDK files along with the demo application.

 	- Demo Application will display the preview of banners.

 	- Documentation explains the integration steps for iOS app integration.


 REQUIREMENTS:

 	1. XCode - iOS Development Environment

 	2. iOS application you wish to integrate.
 	
 	3. SDK to integrate.


 MAINTAINERS:

	iOS SDK.


 FEATURES:

	- Displays the banner by fetching the information from parameters and auto refresh for banner display

	- Additional Device information is fetched for tracking purpose.

	- Performance improvement
	
	- Supports for latest version of iOS
