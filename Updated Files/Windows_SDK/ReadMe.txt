 INTRODUCTION:

 	- Windows SDK version 1.1 package contains the set of SDK files along with the demo application.

 	- Demo Application will display the preview of banners.

 	- The integration steps for Windows app is explained in the documents.


 REQUIREMENTS:

 	1. Visual Studio 2010 - Windows Phone App Development Environment
 	
 	2. Silverlight windows application you wish to integrate.
 	
 	3. SDK to integrate.


 MAINTAINERS:

	WindowsSDK.


 FEATURES:

	- Displays the banner by fetching the information form parameters

	- Additional Device information is fetched for tracking purpose.

	- Performance improvement
	
	- Supports for windows 7 silverlight applications
